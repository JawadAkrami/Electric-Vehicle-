## EV Adoption Trends Dashboard

**From raw data → Python → Power BI insights**

### Overview

EVs are growing fast—and not evenly. In this project, I analyzed Washington State EV data using Python, then turned it into a clean, interactive Power BI dashboard. The goal was simple: make the trends easy to see and understand.

---

### Python Analysis

In the Python phase, I explored and cleaned the dataset to make it usable and meaningful. This included handling missing values, standardizing fields, and breaking the data down by year, model, and location. The focus wasn’t just cleaning—it was understanding what actually matters, like growth trends, top models, and where adoption is happening.

---

### Power BI Dashboard

In Power BI, I turned those insights into a simple, interactive story. The dashboard highlights key metrics like total EVs, leading models, and average range, along with visuals showing growth over time and regional concentration. The goal was clarity—so anyone can quickly understand what’s going on without digging into the raw data.

---

## Key Takeaways

EV adoption really takes off after 2020. Tesla dominates—especially the Model Y and Model 3—while other models lag behind. Adoption is heavily concentrated in urban areas like Seattle, with King County far ahead of others. Range varies quite a bit, with a few models leading, but many balancing cost and performance. There’s also a noticeable chunk of missing eligibility data.
